# Portfolio Content Guide

## What is real vs demo?

The portfolio uses three project labels:

### Real Operations Project
Projects based on real work experience, such as the Alaris PM tracking and inventory reconciliation workflow.

### Academic Project
Projects from biomedical engineering or business analytics coursework, such as DRIPS monitoring, medical imaging, and analytics coursework.

### Portfolio Demo
Projects created to demonstrate capability. These should use anonymized or sample data only.

## Screenshot suggestions

Add screenshots only after anonymizing:

- PM dashboard with sample data
- Duplicate asset highlight example using fake asset tags
- Missing inventory alert example using fake asset tags
- Power BI dashboard using mock healthcare operations data
- Workflow diagram for Health IT device validation

## Do not publish

- Real hospital asset tags
- Internal file names or full paths
- Patient-related information
- Employee names
- Internal hospital screenshots unless approved
- Proprietary spreadsheets
