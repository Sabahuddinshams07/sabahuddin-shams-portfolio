# Step-by-Step Publishing Process

## 1. Create the repository

1. Go to GitHub and sign in.
2. Click the **+** icon in the top-right.
3. Click **New repository**.
4. Repository name:

```text
sabahuddin-shams-portfolio
```

5. Select **Public**.
6. Do not add README, license, or .gitignore.
7. Click **Create repository**.

## 2. Upload files

1. Open this downloaded folder.
2. Upload these files into the repository:
   - `index.html`
   - `README.md`
   - `PUBLISHING_STEPS.md`
   - `PORTFOLIO_CONTENT_GUIDE.md`
3. Commit message:

```text
Initial portfolio upload
```

4. Click **Commit changes**.

## 3. Turn on GitHub Pages

1. Inside the repository, click **Settings**.
2. On the left side, click **Pages**.
3. Under **Build and deployment**, choose:

```text
Source: Deploy from a branch
```

4. Branch:

```text
main
```

5. Folder:

```text
/root
```

6. Click **Save**.

## 4. Open your portfolio

Wait 2–5 minutes.

Your final portfolio link should be:

```text
https://sabahuddinshams07.github.io/sabahuddin-shams-portfolio
```

## 5. Add to LinkedIn

1. Open LinkedIn.
2. Go to your profile.
3. Click **Add profile section**.
4. Click **Recommended**.
5. Click **Add featured**.
6. Choose **Add a link**.
7. Paste:

```text
https://sabahuddinshams07.github.io/sabahuddin-shams-portfolio
```

8. Title:

```text
Sabahuddin Shams | Health IT & Biomedical Operations Portfolio
```

9. Click **Save**.

## 6. Check mobile view

Open the link on your phone and laptop. The portfolio is responsive, so it should adjust automatically.
