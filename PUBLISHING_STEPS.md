# Step-by-Step Publishing Process

## 1. Create repository
1. Go to GitHub.
2. Click + at the top-right.
3. Click New repository.
4. Repository name:
   sabahuddin-shams-portfolio
5. Choose Public.
6. Click Create repository.

## 2. Upload files
1. Open this folder.
2. Upload these files to GitHub:
   - index.html
   - README.md
   - PUBLISHING_STEPS.md
3. Commit message:
   Initial high-fidelity portfolio upload
4. Click Commit changes.

## 3. Turn on GitHub Pages
1. Open your repository.
2. Click Settings.
3. Click Pages.
4. Source:
   Deploy from a branch
5. Branch:
   main
6. Folder:
   /root
7. Click Save.

## 4. Open website
Wait 2–5 minutes.
Your link will look like:

https://YOUR-GITHUB-USERNAME.github.io/sabahuddin-shams-portfolio

## 5. Add to LinkedIn
1. Go to LinkedIn profile.
2. Add profile section.
3. Recommended → Add featured.
4. Add link.
5. Paste your GitHub Pages link.
6. Title:
   Sabahuddin Shams | Biomedical Operations & Healthcare Analytics Portfolio
