# Update Existing GitHub Portfolio

## Replace the portfolio file
1. Open your GitHub repository.
2. Click `index.html`.
3. Click the pencil / edit icon.
4. Delete all the old content.
5. Paste the new content from this updated `index.html`.
6. Scroll down.
7. Commit message:
   Update portfolio with detailed operations, vendor, and Health IT content
8. Click Commit changes.

## Alternative method
1. Click Add file → Upload files.
2. Upload the new `index.html` and `README.md`.
3. Commit the changes.

## Wait for GitHub Pages
After commit, wait 1–5 minutes and refresh your live site.
